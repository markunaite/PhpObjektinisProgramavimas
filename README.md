# PHP objektinis programavimas

* PHP web puslapis

* PHP My Admin
    * http://localhost:8090
* Mysql Duomenų bazė

## Reikalavimai

* docker
* docker-compose

## Paleidimas
 
```
docker-compose up -d
```

## Sustabdymas

```
docker-compose stop
```

# Konteinerių ištrynimas

```
docker-compose down
```
